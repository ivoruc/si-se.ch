Tutorium I: Einführung in die Entwicklung auf der Microsoft Azure Plattform

Markus Leder, Principal Consultant, Zühlke

Microsoft Azure ermöglicht Ihnen das schnelle Erstellen, Bereitstellen und Verwalten von Anwendungen auf einer Cloud Plattform in einem globalen Netzwerk von Rechenzentren. Anwendungen können mit beliebigen Betriebssystemen, Sprachen oder Tools erstellt werden. Markus Leder gibt Ihnen einen Überblick über die Plattform und deren Dienste. Danach werden Sie eine einfache Web Applikation in Visual Studio mit C# erstellen - inklusive Datenbank - und in Microsofts PaaS Cloud "deployen". In diesem Tutorial werden wir auch diskutieren, welche architektonischen Überlegungen für eine erfolgreiche Entwicklung wichtig sind.

Voraussetzungen: persönliches Notebook mit Internet Zugang und Oracle VM VirtualBox installiert (Image für ca. 4GB RAM und 30GB HD). Falls möglich kostenloses Testkonto von Windows Azure. Kenntnisse in C# und .NET werden nicht vorausgesetzt.

Tutorium II: Mobile front-end for Java back-ends - the cloud solution

Sacha Labourey, CEO, CloudBees

Presentation of the CloudBees platform and how it can help Java developers and enterprises become much more productive while leveraging their existing Java assets. It will be demonstrated how to build and test a mobile application (iOS and Android) that interacts with a Java back-end, with no need for any infrastructure or software, but a laptop.
